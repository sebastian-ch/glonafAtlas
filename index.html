The GloNAF Interactive Atlas

This interactive atlas visualizes the GloNAF (Global Naturalized Alien Flora) dataset. The GloNAF is a living database project about alien plant species. This data compendium includes over 13,900 taxa and covers 1,029 different regions, including 381 islands (as of 10.08.2020). It is based on 210 different sources.


The GloNAF dataset version 1.2 and the published paper can be found here. Please note, if you would like to use the GloNAF data we encourage users to get in touch with the GloNAF core team in order to get the most up-to- date version of the required data in GloNAF.
